Zomato Data Analysis 🍽️📊

This project focuses on analyzing Zomato restaurant data to uncover valuable insights about customer preferences, restaurant trends, and market behavior. Using Python, Pandas, NumPy, Matplotlib, and Seaborn, the dataset is explored, cleaned, and visualized to identify patterns in ratings, cuisines, locations, and pricing.

📌 Project Objectives

Perform data cleaning and preprocessing on raw Zomato data.
Analyze restaurant ratings and voting patterns.
Explore popular cuisines and restaurant types.
Identify cost distribution for two people.
Compare online vs offline ordering trends.
Discover location-based restaurant insights.
🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📂 Dataset Features

The dataset includes:

Restaurant Name
Location
Cuisine Type
Average Cost for Two
Ratings
Votes
Online Order Availability
Table Booking Availability\

📈 Key Insights

Most preferred restaurant categories by customers.
Correlation between ratings and number of votes.
Popular locations with the highest restaurant density.
Impact of online ordering on ratings.
Cost trends across different cuisines.
